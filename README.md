# 🚀 SocialViralAI

AI-powered web app for analyzing YouTube video trends and identifying viral content.

## 🔥 Features

* Search videos by keyword
* Analyze trending videos using custom ViralScore
* AI-generated insights for content strategy
* Shorts-style UI (vertical videos)
* Responsive grid layout

## 🧠 Tech Stack

* ASP.NET Core MVC
* C#
* YouTube Data API v3
* HTML / CSS
* OpenAI API

## 💡 Key Idea

Instead of relying on YouTube ranking, this app calculates its own viral score:

ViralScore = (Views / 1000) + (Likes \* 5)

This helps identify trending content faster.
